# Roadmap Inicial

## Fase 1: Base

- definir nombre y estilo visual
- elegir stack
- crear repositorio
- acordar flujo de ramas

## Fase 2: Contenido

- reunir bios del equipo
- seleccionar proyectos
- escribir descripciones orientadas a cliente
- definir servicios y diferenciadores

## Fase 3: Desarrollo

- montar layout principal
- construir secciones del home
- hacer responsive mobile y desktop
- agregar SEO básico

## Fase 4: Conversión

- conectar formulario o CTA externo
- agregar métricas o analítica
- revisar textos y velocidad de carga

## Fase 5: Publicación

- desplegar
- probar dominio
- compartir y recoger feedback
